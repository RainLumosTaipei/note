概述 ​

jQuery是一个快速、简洁的JavaScript框架，是继Prototype之后又一个优秀的JavaScript代码库（框架）于2006年1月由John Resig发布。

jQuery设计的宗旨是write Less，Do More，即倡导写更少的代码，做更多的事情。它封装JavaScript常用的功能代码，提供一种简便的JavaScript设计模式，优化HTML文档操作、事件处理、动画设计和Ajax交互。

jQuery的核心特性可以总结为：具有独特的链式语法和短小清晰的多功能接口；具有高效灵活的CSS选择器，并且可对CSS选择器进行扩展；拥有便捷的插件扩展机制和丰富的插件。

语法 ​

所有代码必须在document加载后添加，可以将其放在html代码后，也可以用以下代码包围：

js
$(document).ready(function () {
// jQuery绑定DOM元素，处理事件等
})

DOM ​

js
// 获取class类名的所有元素
$(".class") 

// 获取class类名的第一个元素
$(".class:eq(0)") 
$(".class").eq(0)

// 获取id元素
$("#id")

// 获取div标签
$("div")

// 修改类名
$(".class").addClass("newClass")
$(".class").removeClass("newClass")
// 二合一，有则删除，无则添加
$(".class").toggleClass("newClass")

// 添加子代元素
let child = $("<div>666</div>")
// 最后方添加
$(".class").append(child)
// 最前方添加
$(".class").prepend(child)
// 替换
$('ul>li').replaceWith('<p>我是段落</p>');

// 获取父元素
let p = $('.box').parent()[0]
// 获取子元素
let child = $('ul').children();
// 获取前一个兄弟元素
let p = $('.box').prev()[0]
// 获取后一个兄弟元素
let p = $('.box').next()[0]

// 删除
// 清空子代
$(".class").empty()
// 清空特定
$(".class").remove()

ajax ​

js
// get
$.ajax({
    url: '/user/login',
    type: 'get',
    data: {
        username: 'zhangsan',
        password: '123456'
    },
    dataType: 'text',
    success: function (response) {
        alert(response);
    },
    error: function (response) {
        alert(response);
    }
});

// post
$.ajax({
    url: '/user/login',
    type: 'post',
    data: {
        username: 'zhangsan',
        password: '123456'
    },
    dataType: 'text',

        alert(response);
