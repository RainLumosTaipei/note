概述 ​

Vue3是JavaScript框架Vue.js的第三个主要版本，由尤雨溪领导的Vue团队于2020年9月15日发布。相较于Vue 2.x，Vue3进行了深度重构和优化，并引入了一系列新的特性以提升性能、易用性和灵活性。

ad-abstract

- 发布时间：2020年9月15日

- 官方文档：<https://cn.vuejs.org/guide/introduction.html>

- 仓库地址：<https://github.com/vuejs/vue-next>

以下是Vue3的主要更新点：

性能提升

  • 打包大小显著减少（41%），这有助于提高应用加载速度。

  • 初次渲染速度加快了约55%，更新渲染速度提高了大约133%。

  • 内存占用减少了54%，意味着更高效的内存管理和资源利用。

新架构与API变化

  • Composition API： 引入了全新的Composition API，它允许开发者在一个集中式的逻辑函数setup()中组织组件的状态和行为，相比Options API提供了一种更加灵活且易于重用代码的方式。

  • 响应式系统改进： 使用ES6的Proxy代替Object.defineProperty实现更完整的响应式代理，增强了对对象属性添加/删除等操作的监听能力。

  • Tree-Shaking支持： 通过更好的模块化设计，Vue3能够更好地支持treeshaking，从而减小生产环境下的打包体积。

  • 组件生命周期调整: Vue3移除了部分生命周期钩子如beforeCreate和created，并推荐在setup()中处理相关逻辑。 引入了一些新的生命周期钩子，例如onMounted、onUpdated、onBeforeUnmount等。

  • 其他新特性与优化: ref和reactive等核心API用于管理组件状态。 Suspense组件用于异步内容的延迟加载。 Teleport组件用来将DOM节点“传送”到页面的其他位置。 支持TypeScript编写，官方提供了丰富的类型定义文件。

  • 工具链支持: Vue CLI 和 Vite 的更新，Vite 提供了更快的开发体验，利用原生ES模块实现快速热更新。

目录结构 ​

public 文件夹中包括 ico 和 index.html 入口文件

vue.config.js 为项目配置文件，可配置项目打包、运行等参数

src 文件夹下为 Vue 核心工程

ad-note
- `App.vue` 为根组件，其余组件由此引入
- `main.js` 为全局 js 文件
- `assets` 存放图片等静态资源
- `router` 存放路由配置文件
- `components` 存放复用的 Vue 组件
- `store` 存放 `Vuex` 或 `Pinia` 状态管理
- `views` 存放视图页面组件

路由 ​

npm 安装 vue-router 管理路由

powershell
npm install vue-router

修改 main.js 如下：

javascript
import { createApp } from 'vue'
import App from './App.vue'
import router from './router/index' //导入路由表

const app = createApp(App);
app.use(router).mount('#app'); //使用路由导航

之后在 src 文件夹下新增 router 目录，在其下创建 index.js 配置路由:

javascript
//引入组件，@代表根目录指向 src
import { createRouter, createWebHistory } from 'vue-router'

//配置路由表
const router = new createRouter({
  history: createWebHistory(), 
  const router = new createRouter({
  history: createWebHistory(),
  routes: [
    {
      path: "/",
      name: "home",
      component: ()=> import("@/views/Home.vue"),
      meta: {
        title: "网页 | 首页",
      },
    },
    {
      path: "/login",
      name: "login",
      component: ()=> import("@/views/Login.vue"),
      meta: {
        title: "网页 | 登录",
      },
    }
  ],
});
})

//每次路由跳转前验证信息
router.beforeEach((to, from, next) => {
  if (to.meta.isAuth) {
    //全局变量保存登录状态
    if (sessionStorage.getItem("login") === 1) {  
      next()  
    } 
    else {
      alert('抱歉，您无权限查看！');
      //跳转登录页面
      next('/login');  
    }
  } 
  else {
    next()
  }
})

//每次路由跳转后更新标题
router.afterEach((to, from) => {
  document.title = to.meta.title || "网页 | 欢迎"
})
export default router;

组件传值 ​

Mitt ​

mitt可以用于任意两个组件之间的通信，解决了非父子关系的参数传递：

  • 安装

npm 安装 mitt 事件总线

powershell
npm install mitt

修改 main.js 如下：

javascript
import { createApp } from 'vue'
import App from './App.vue'
import mitt from 'mitt' //导入mitt

const app = createApp(App);
//设置$bus代替mitt
app.config.globalProperties.$bus = new mitt(); 
app.mount('#app');
  • 发送数据组件:
javascript
import { getCurrentInstance } from "vue";
const { proxy } = getCurrentInstance();

let param = ref();
proxy.$bus.emit("busname", param.value);
  • 接受数据组件:
javascript
import { getCurrentInstance } from "vue";
const { proxy } = getCurrentInstance();

//必须在加载之前传值，使用onMounted
let curParam = ref();
onMounted(() => {
  proxy.$bus.on("busname", (param) => {
    curParam.value = param; 
  });
});

vue.config.js 配置 webpack 打包，并使用gzip压缩：

shell
npm install compression-webpack-plugin
js
const { defineConfig } = require("@vue/cli-service");
const webpack = require('webpack');

// 安装压缩
const CompressionPlugin = require('compression-webpack-plugin')

module.exports = defineConfig({
  lintOnSave: false,
  transpileDependencies: true,
  configureWebpack: {
    plugins: [
	  // 取消警告
      new webpack.DefinePlugin({
        __VUE_OPTIONS_API__: JSON.stringify(true),
        __VUE_PROD_DEVTOOLS__: JSON.stringify(false),
        __VUE_PROD_HYDRATION_MISMATCH_DETAILS__: JSON.stringify(false)
      }),
      // 压缩配置
      new CompressionPlugin({
        filename: '[path].gz',
        algorithm: 'gzip',
        test: /\.js$|\.css$|\.html$/,
        threshold: 10240,
        minRatio: 0.8,
        deleteOriginalAssets: false
      }),
    ]
  },
})

js-cookie ​

  • 安装
shell
npm install js-cookie
  • 封装
js
import Cookies from "js-cookie";

export const getCookie = (key) => {
  //字符化
  let json = JSON.stringify({ [key]: Cookies.get(key) });
  //转化为对象
  let obj = JSON.parse(json);
  return obj;
};

export const setCookie = (key, value) => {
  //匹配 \
  let reg1 = /\\/g;
  //匹配开始和结束的"
  var reg2 = /^['|"](.*)['|"]$/;
  //字符化
  let json = JSON.stringify(value);
  json = json.replace(reg1, "");
  json = json.replace(reg2, "$1");
  json = json.replace(reg2, "$1");
  //3天后过期
  return Cookies.set(key, json, { expires: 3 });
};

export const removeCookie = (key) => {
  return Cookies.remove(key);
};
  • 更新
