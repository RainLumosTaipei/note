C 语言问题集 ​

以下大多数内容来自 https://c-faq.com/, 这里包含了很多 C 语言的问题, 阅读这些问题和解答可以帮助 我们很好的学习 C 语言.相关书籍可以阅读"你必须知道的 495 个 C 语言问题"

这些问题除了一些语法问题,其余大多不会在其他语言中遇到.但正是这些独特的问题可以帮助我们了解实际的情况. 这里有很多实际的问题,需要仔细地考虑内存以及特定的机器上的实现.通过思考这些问题,我们理解每一条语句的实际 作用.

* 优先级 ​

括号优先级高于指针

int (*fp)(int) 是函数指针

int* fp(int) 是函数

关于指针 * ​

char *p, q 并没有声明两个指针, 而是声明一个指针和一个字符

因为char是基本类型, 而不是char*, *应当作为标识符的一部分, 这也说明了为什么指针运算符通常与标识符在一起,而不是与基本类型合并在一起.

extern ​

extern 只在变量声明时有作用,他暗示这个变量的定义可能不在当前文件中,他可以出现在变量声明中,也可以出现在函数声明中. 但是如果不写也不会有问题,与直接声明相同,链接器会处理.

typedef ​

typedef 可以正确处理指针类型,而仅仅靠宏 define 是做不到的

c
#define string char *
string p, q; // not expected

typedef char *string;
string p, q;

typedef 和 struct ​

在使用 typedef 定义 struct 时,无法在 typedef 之前使用类型别名:

c
typedef struct node{
    int a;
    struct node *next;  // this is ok
} *nodeptr;

typedef struct node{
    int a;
    nodeptr next;  // nodeptr has not defined
} *nodeptr;

相互引用的 struct ​

struct A 和 struct B 是相互引用的, 那么在 struct A 中声明一个 struct B 的指针, 在 struct B 中声明一个 struct A 的指针,并不会导致循环引用

重点在于理解变量具有声明和定义两个阶段.

c
// this is acceptable
// the previous declare is not needed

// struct A;
// struct B;

struct A{
       int a;
       struct B *p;
   };

struct B{
    int b;
    struct A *p;
};

typedef int(*funcptr)();    /*generic function pointer */

初始化 ​

  • 静态变量

不在任何函数作用域的变量,即全局变量或者 static 变量,会自动赋初值 0

静态变量初始化时无法调用函数

  • 非静态变量

在函数作用域的变量, 或者 malloc 分配的变量,默认为垃圾值

字符串初始化 ​

c
char *p = "hello";
p[2] = 'a';  // not allowed

hello 被存储在只读内存中,所以不能修改

成员函数 ​

可以将函数指针嵌入 struct 中,从而实现成员函数

字符串大小 ​

c
struct name {
	int namelen;
	char namestr[1];
};

可以在初始化时改变数组的大小:

c
#include <stdlib.h>
#include <string.h>

struct name *makename(char *newname)
{
	struct name *ret =
		malloc(sizeof(struct name)-1 + strlen(newname)+1);
				/* -1 for initial [1]; +1 for \0 */
	if(ret != NULL) {
		ret->namelen = strlen(newname);
		strcpy(ret->namestr, newname);
	}

	return ret;
}

也可以如此声明:

c
#define MAXSIZE 100

struct name {
	int namelen;
	char namestr[MAXSIZE];
};

如果使用字符指针,则数组内容和长度不在同一片区域,且释放内存需要两次 free

c
#include <stdlib.h>
#include <string.h>

struct name {
	int namelen;
	char *namep;
};

struct name *makename(char *newname)
{
	struct name *ret = malloc(sizeof(struct name));
	if(ret != NULL) {
		ret->namelen = strlen(newname);
		ret->namep = malloc(ret->namelen + 1);
		if(ret->namep == NULL) {
			free(ret);
			return NULL;
		}
		strcpy(ret->namep, newname);
	}

	return ret;
}

可以使用如下技巧将两片内存区域和在一起:

c
struct name *makename(char *newname)
{
	char *buf = malloc(sizeof(struct name) +
				strlen(newname) + 1);
	struct name *ret = (struct name *)buf;
	ret->namelen = strlen(newname);
	ret->namep = buf + sizeof(struct name);
	strcpy(ret->namep, newname);

	return ret;
}

sizeof ​

sizeof 返回的值可能大于 struct 的大小,需要考虑字节对齐和填充

使用 offsetof 可以得知域的偏移量

c
#include <stddef.h>

struct name {
	int namelen;
}

printf("%d\n", offsetof(struct name, namelen));

或者使用宏:

c
#define offsetof(type, member) \
	((size_t) &((type *)0)->member) /* 0 is a null pointer */

printf("%d\n", offsetof(struct name, namelen));

以上对于空指针内部表示为 0 时可以正确计算偏移,保险起见应采用减法:

c
#define offsetof(type, f) ((size_t) \
		((char *)&((type *)0)->f - (char *)(type *)0))
```

使用特定地址可能会导致非法访问:

```c
#define offsetof(type, member) \
	((size_t) &((type *)1)->member - 1)

返回 struct ​

C 可以返回结构,但不推荐.返回结构时会包含一个隐含的指针.

结构的定义应该以分号结尾,有可能会出现如下错误:

c
struct name {
	int namelen;
	char namestr[];
}

main(){
}

上述代码会被认为 main 函数返回一个结构

union ​

联合取最大字段长度,特点在于可以用不同方式读取字段

枚举 ​

枚举的大小为 int,无法手动调整,所以会有类型转换之类的警告. 可以使用宏定义来代替枚举

位域 ​

可以使用位域指定字段的大小,只能在结构中使用:

c
struct record {
	char *name;
	int refcount : 4;
	unsigned dirty : 1;
};

i++ ​

C
int i = 0;
i = i++;

无法确定 i 的值,因为编译器实现无法确定 注意与 i = i + 1 的区别

C++ 中应该优先使用 ++i, C 中无所谓

交换 ​

c
int a = 1, b = 0;

a ^= b ^= a ^= b;

不具备可移植性,因为编译器实现无法确定

整数除法 ​

C
double degC, degF;
degC = 5 / 9 * (degF - 32);

整数除法会得到 0,因为 5 / 9 = 0

?: ​

C
a ? b = c : d

隐含了中部表达式的括号

符号类型转换 ​

可能存在两种类型转换方式:

  1. 值转换,即保证转换后值保持不变
  2. 无符号转换,统一转换为无符号类型

两种转换方式会导致不同的结果.对于同时涉及有符号和无符号的表达式, 应进行显示类型转换来避免这种异常

C
unsigned short us = 10;
int i = -5;
if(i > us)
	printf("whoops!\n");
C
unsigned char uc = 0x80;
unsigned long ul = 0;
ul |= uc << 8;
printf("0x%lx\n", ul);

*p++ ​

*p++ 会将 p 自增并返回自增之前的旧值,等价于*(p++)

数组指针 ​

c
int arr[10];
int* p;

p = arr;

// 正确用法
printf("%d\n", *(p + 3));
// 错误用法
printf("%d\n", *(p + 3 * sizeof(int)));
