简介 ​

一个语言的语法并不能决定一个语言的类型。在现代语言的范围下，语法应该吸收前人的优秀之处，尽可能将编译器的功能完善，简化语法且已读。下面列举几个可以学习的地方：

  • 行末无需分号
  • 变量类型可以自动推断
  • 类型后置

Go的一些设计理念：https://go.dev/talks/2012/splash.article 例如：优化编译，组合非继承，简洁语法，自动化工具，最小依赖，包管理，垃圾收集，并发，错误寻常化

基本语法 ​

变量声明 ​

在函数体内部，一般采用:= 声明局部变量，可以根据赋值自动推断类型。注意，局部变量声明后必须使用。

go
func add(a, b int) int {
	a := 1
	b, c := 2
}

全局变量的声明必须显式的表示，使用var关键字，且声明后可以不使用。

go
var a, b int
a = 1
b = 2
var c = 1

// 交换a,b的值
a, b = b, a

此外，_可用于表示忽略值的变量，只可以写入，不可以读取。例如忽略函数的部分返回值。

go
func main(){
	_, a := add(1,2)
}

常量声明 ​

常量可以集中声明，且可以自动递增。

go
func main() {
    const (
            a = iota   //0
            b          //1
            c          //2
            d = "ha"   //"ha" iota += 1
            e          //"ha" iota += 1
            f = 100    //100  iota += 1
            g          //100  iota += 1
            h = iota   //7
            i          //8
    )
    fmt.Println(a,b,c,d,e,f,g,h,i)
    // 0 1 2 ha ha 100 100 7 8
}

函数声明 ​

函数声明包括参数和返回值，返回值写在最后，且可以有多个。

go
func swap(x, y string) (string, string) {  
   return y, x  
}

函数也可以在函数内部声明，此时为匿名函数。

go
func main() {  
    add := func(a, b int) int {  
       return a + b  
    }  
    c := add(1, 2)  
}

函数名前可以添加其所属的类，此时为类的方法。

go
/* 定义结构体 */

  radius float64  
}  
  
// 该 method 属于 Circle 类型对象中的方法  
func (c Circle) getArea() float64 {  
  return 3.14 * c.radius * c.radius  
}

channel ​

通道用于在线程之间传递数据；

  • 若没有缓冲区，则数据取出之前发送方会阻塞；如果接收操作先执行，接收方的goroutine将阻塞，直到另一个goroutine在该通道上发送一个值。无缓冲通道也被称为同步通道。
  • 若有缓冲区，则数据放入缓冲区后发送方继续运行，直到缓冲区占满。缓冲通道为空时，尝试接收数据时发生阻塞。
go
// 无缓冲区
ch := make(chan int)
// 缓冲区大小为 10
ch := make(chan int 10)

举例：

go
func sum(s []int, c chan int) {
    sum := 0
    for _, v := range s {
        sum += v
    }
    c <- sum 
}

func main() {
    s := []int{7, 2, 8, -9, 4, 0}
    c := make(chan int)
        
    go sum(s[:len(s)/2], c)
    go sum(s[len(s)/2:], c)
        
    x, y := <-c, <-c 
    fmt.Println(x, y, x+y)
}

通道可以用close主动关闭，被关闭的信道会禁止数据流入, 是只读的。我们仍然可以从关闭的信道中取出数据，但是不能再写入数据了。

go
ch := make(chan int, 3)
ch <- 1
ch <- 2
ch <- 3
 
// 显式地关闭信道
close(ch)
 
for v := range ch {
    fmt.Println(v)
}

使用range可以取出通道缓冲区中所有数据，只有通道被关闭时，才不会阻塞。

go
func main() {
    ch1 := make(chan int)
    ch2 := make(chan int)
    
    // 开启goroutine将0~100的数发送到ch1中
    go func() {
        for i := 0; i < 100; i++ {
            ch1 <- i
        }
        close(ch1)
    }()
    
    // 开启goroutine从ch1中接收值，并将该值的平方发送到ch2中
    go func() {
        for {
            // 通道关闭后再取值 ok = false
            i, ok := <-ch1 
            if !ok {
                break
            }
            ch2 <- i * i
        }
        close(ch2)
    }()
    
    // 在主goroutine中从ch2中接收值打印
    for i := range ch2 { // 通道关闭后会退出 for range 循环
        fmt.Println(i)
    }
}

select ​

select 语句使得一个 goroutine 可以等待多个通信操作。select 会阻塞，直到其中的某个 case 可以继续执行。注意，通道关闭后仍然可以被select选中读取，此时为零值。

go
package main

import (
   "log"
   "time"
)

func main() {
   // 设置log参数
   log.SetFlags(log.Lshortfile | log.LstdFlags)

   var ch1 = make(chan int, 5)
   var ch2 = make(chan int, 5)

   // 直接写一个匿名函数 向通道中添加数据
   go func (){
      var num = 1
      for {
         ch1 <- num
         num += 1
         time.Sleep(1 * time.Second)
      }
   }()

   go func (){
      var num = 1
      for {
         ch2 <- num
         num += 1
         time.Sleep(1 * time.Second)
      }
   }()

   for {
	  // 读取数据
      select {
      case num := <-ch1:
         log.Printf("read ch1 data is  %d\n", num)

      case num := <-ch2:
         log.Printf("read ch2 data is: %d\n", num)

      default:
         log.Printf("ch1 and ch2 is empty\n")
          // 休息 1s 再读
         time.Sleep(1 * time.Second)
      }
   }
}

shell
go mod init example.com/hello

shell
go list -m -versions rsc.io/sampler

shell
go get rsc.io/sampler@v1.3.1

标准库 ​

log ​

标准日志有3种类型：

go
log.Println()
log.Panicln() // recover()
log.Fatalln() // exit(1)

自定义日志输出格式：

go
var (  
    Info  *log.Logger  
    Error *log.Logger  
)  
  
func main() {  
	// 打开日志文件
    file, err := os.OpenFile("error.log", os.O_CREATE|os.O_WRONLY|os.O_APPEND, 0666)  
    if err != nil {  
       log.Fatalln("Failed to open error log file:", err)  
    }  
	// 创建 Logger
    Info = log.New(os.Stdout, "INFO: ", log.Ldate|log.Ltime|log.Lshortfile)  
    Error = log.New(io.MultiWriter(file, os.Stderr), "ERROR: ", log.Ldate|log.Ltime|log.Lshortfile)  
	// 调用日志
    Info.Println("info66")  
    Error.Println("Error9999")  
}

Logger结构：

go
type Logger struct {  
    outMu sync.Mutex  
    
    out   io.Writer // destination for output  
    prefix    atomic.Pointer[string] // prefix on each line to identify the logger (but see Lmsgprefix)  
    flag      atomic.Int32           // properties  
    isDiscard atomic.Bool  
}  
  
// New creates a new [Logger]. The out variable sets the  
// destination to which log data will be written.  
// The prefix appears at the beginning of each generated log line, or  
// after the log header if the [Lmsgprefix] flag is provided.  
// The flag argument defines the logging properties.  
func New(out io.Writer, prefix string, flag int) *Logger {  
    l := new(Logger)  
    l.SetOutput(out)  
    l.SetPrefix(prefix)  
    l.SetFlags(flag)  
    return l  
}

日志格式常量定义：

go
const (  
    Ldate         = 1 << iota     // the date in the local time zone: 2009/01/23  
    Ltime                         // the time in the local time zone: 01:23:23  
    Lmicroseconds                 // microsecond resolution: 01:23:23.123123.  assumes Ltime.  
    Llongfile                     // full file name and line number: /a/b/c/d.go:23  
    Lshortfile                    // final file name element and line number: d.go:23. overrides Llongfile  
    LUTC                          // if Ldate or Ltime is set, use UTC rather than the local time zone    Lmsgprefix                    // move the "prefix" from the beginning of the line to before the message  
    LstdFlags     = Ldate | Ltime // initial values for the standard logger  
)

http ​

http服务器需要实现ServeHTTP接口：

go
type Engine struct{}  

// 必须实现此方法，用于对请求构造响应
func (engine *Engine) ServeHTTP(w http.ResponseWriter, req *http.Request) {  
    switch req.URL.Path {  
    case "/":  
       fmt.Fprintf(w, "URL.Path = %q\n", req.URL.Path)  
    case "/hello":  
       for k, v := range req.Header {  
          fmt.Fprintf(w, "Header[%q] = %q\n", k, v)  
       }  
    default:  
       fmt.Fprintf(w, "404 NOT FOUND: %s\n", req.URL)  
    }  
}  
  
func main() {  
    engine := new(Engine)  
    // 端口号
    http.ListenAndServe(":9999", engine)
    /* 单独写法
	http.HandleFunc("/", indexHandler)  
	http.HandleFunc("/hello", helloHandler)
	*/
}
