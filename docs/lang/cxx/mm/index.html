C++ 内存管理 ​

C++内存管理函数有以下四个层次, 并且上层逐层调用下层:

  1. std::allocator

标准库的 allocator, 用于对容器进行内存管理

  1. new, new[], new(), ::operator new()

本质为 operator new 的各种变体, 经过重载就可以用来实现多种内存管理方式, 这里是自定义代码的入手点

  1. malloc, free

基本的 C 库函数,其内部原理复杂,需要深入研究

  1. OS API: mmap, HeapAlloc, VirtualAlloc

操作系统层面的内存分配函数

TIP

内存管理有几个要点:

  1. 分配速度快,考虑减少 malloc 次数
  2. 占用空间小,尽可能压缩对象分配占用的额外空间
  3. 减少内存碎片
  4. 便于统一管理和回收

alloctor ​

alloctor 用于标准库容器分配内存 , 以下我们逐步分析 alloctor 的实现 . 对于源码 , 应该了解其发展历程 , 从演进中可以看出思想的变化 . 现有的实现不一定是最佳的 , 有些过去的代码仍然有借鉴意义 , 需要全面的认识 .

per-class allocator ​

这里模拟一个简单的分配器, 适用于单个类的内存分配.

内部管理一个对象池, 一次申请多个对象的内存, 减少 malloc 次数, 提高了速度, 也减少了 malloc 带来的额外空间开销.

将所有未分配的内存空间串成一个单向链表, 当释放内存时, 直接将释放的内存插入头部, 下次申请内存时, 从链表头分配内存.

使用 union 可以巧妙的复用字段存放指针.

c++
class Demo{
public:
    union {
        int val;
        Demo* next;
    };

    // 内存池总数目
    static const int tol;
    // 空闲链表头指针
    static Demo* head;

    static void * operator new(size_t);
    static void operator delete (void*, size_t);
};

const int Demo::tol = 512;
Demo* Demo::head = nullptr;
c++
void* Demo::operator new(size_t size) {
    // 首次申请
    if(!head){
        void* p = malloc(tol*size);
        head = static_cast<Demo*>(p);
        // 串成链表
        for (int i = 0; i < tol; ++i) {
            head[i].next = &head[i+1];
        }
    }
    // 分配头部指针
    Demo* res;
    res = head;
    head = head->next;
    return res;
}
c++
void Demo::operator delete (void* p, size_t) {
    // 插入头部
    static_cast<Demo*>(p)->next = head;
    head = static_cast<Demo*>(p);
}

该分配器非常简单, 主要在于使用内存池分配多个对象, 并使用单向链表管理空闲内存. 这是最初步的思想, 是一切思路的起点.

static allocator ​

现在我们考虑将 allocator 应用到多个类中, 由于各个类中 allocator 的实现相似, 因此作成一个单独的类:

c++
class Allocator{
private:
    typedef struct obj{
        struct obj *next;
    } Obj;
public:
    void* allocate(size_t size);
    void deallocate(void* p, size_t size);

    const int tol = 512;
    struct obj* head = nullptr;
};

void* Allocator::allocate(size_t size)  {
    if(!head){
        void* p = malloc(tol*size);
        head = static_cast<Obj*>(p);
        for (int i = 0; i < tol; ++i) {
            head[i].next = &head[i+1];
        }
    }
    Obj* res;
    res = head;
    head = head->next;
    return res;
}

void Allocator::deallocate(void *p, size_t size) {
    static_cast<Obj*>(p)->next = head;
    head = static_cast<Obj*>(p);
}

对于调用的类, 添加一个 static 字段 allocator, 在类中重载 new 和 delete, 使用 allocator 的 allocate 和 deallocate.

c++
class Demo{
public:

    int val;
    static Allocator allocator;

    static void * operator new(size_t size){
        return allocator.allocate(size);
    }
    static void operator delete (void* ptr, size_t size){
        allocator.deallocate(ptr, size);
    }

};

GNU 2.95 ​

在 GNU 2.95 中, allocator 有多个实现, 对于标准库的 allocator, 没有特殊的地方, 直接调用 operator new.
